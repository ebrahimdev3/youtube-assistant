# YouTube Smart Assistant

A lightweight Chrome Extension built with Manifest V3 that improves your YouTube browsing experience.

## Features
- **Watch History Tracking:** Automatically blurs video thumbnails you've already watched so you can focus on new content.
- **Restore Dislikes:** Fetches and displays the dislike counts for videos using the Return YouTube Dislike API.
- **Performance Optimized:** Built with efficient DOM observation and debouncing to ensure zero lag.

## Tech Stack
- JavaScript (Vanilla)
- CSS3
- Chrome Extension API (Manifest V3)
